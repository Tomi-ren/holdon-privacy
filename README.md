# 隐私政策 / Privacy Policy

## 应用名称：坑你没（Hold On）

本应用非常重视用户的隐私与数据安全。

### 1. 信息收集
当用户主动开启相关功能时，本应用可能会使用设备的麦克风进行音频采集，仅用于在本地进行分析与提醒。

### 2. 信息使用
采集的音频数据仅用于应用内部的功能实现，不用于任何其他用途。

### 3. 数据存储
所有音频数据仅保存在用户设备本地，不会上传至任何服务器，也不会与任何第三方共享。

### 4. 数据控制
用户可以随时在应用内停止相关功能，并自行管理或删除本地数据。

### 5. 权限说明
本应用仅在用户明确操作后使用相关权限，不会在后台自动录音。

### 6. 联系方式
如有隐私相关问题，可通过应用内提供的方式联系开发者。

## Privacy Policy (English)

### App Name: Hold On

We value your privacy and are committed to protecting your personal data.

### 1. Data Collection
When the user explicitly enables certain features, the app may access the device microphone to collect audio data for local analysis and reminders only.

### 2. Data Usage
Collected audio data is used solely to provide app functionality and is not used for any other purposes.

### 3. Data Storage
All audio data is processed and stored locally on the user's device. No data is uploaded to any server or shared with third parties.

### 4. User Control
Users can stop the feature at any time and manage or delete local data on their device.

### 5. Permissions
The app accesses permissions only after explicit user action and does not perform background recording automatically.

### 6. Contact
If you have any questions regarding this privacy policy, please contact the developer via the app.
